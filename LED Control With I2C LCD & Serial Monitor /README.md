LED Control with I2C LCD and Serial Monitor


This project demonstrates how to interface an I2C LCD display and an LED with an Arduino Uno. The LCD displays a welcome message, the Serial Monitor shows the current status of the LED, and the LED blinks continuously. The project introduces serial communication, LCD interfacing using the I2C protocol, and digital output control.

---

Hardware Required

- Arduino Uno
- I2C 16×2 LCD Display
- LED
- 220 Ω Resistor
- Breadboard
- Jumper Wires
- USB Cable

---

Software Required

- Arduino IDE
- LiquidCrystal_I2C Library

---

Theory

Arduino Uno

Arduino Uno is an open-source microcontroller board based on the ATmega328P. It is widely used for learning embedded systems, robotics, and automation.

---

Arduino IDE

The Arduino IDE is used to write, compile, and upload Arduino sketches. It also provides tools such as the Serial Monitor for debugging and monitoring program execution.

---

I2C LCD Display

An I2C LCD is a 16×2 character display that communicates with the Arduino using only two data lines: SDA and SCL. This reduces wiring complexity compared to a standard LCD interface.

---

Serial Monitor

The Serial Monitor is used to display messages sent by the Arduino. In this project, it indicates when the LED is turned ON and OFF and confirms the start of the experiment.

---

LED Interfacing

An LED connected through a 220 Ω resistor to a digital output pin is controlled using HIGH and LOW signals generated by the Arduino.

---

Circuit Connections

| Arduino Uno Pin | Component |
|-----------------|-----------|
| Pin 12 | LED Anode (through 220 Ω resistor) |
| GND | LED Cathode |
| 5V | LCD VCC |
| GND | LCD GND |
| A4 (SDA) | LCD SDA |
| A5 (SCL) | LCD SCL |

Working Principle

When the Arduino starts, the LCD is initialized and displays "Arduino". At the same time, the Serial Monitor prints "Experiment Started". The Arduino then repeatedly turns the LED ON and OFF every second while displaying the corresponding status ("LED ON" and "LED OFF") in the Serial Monitor.

---

Procedure

1. Connect the LED to digital pin 12 through a 220 Ω resistor.
2. Connect the I2C LCD to the Arduino using the SDA and SCL pins.
3. Open the Arduino IDE.
4. Install the LiquidCrystal_I2C library if it is not already installed.
5. Upload the program to the Arduino Uno.
6. Open the Serial Monitor and observe the messages.
7. Observe the LED blinking and the LCD displaying the welcome message.

---

Expected Outcome

- The LCD displays "Arduino".
- The LED blinks continuously at one-second intervals.
- The Serial Monitor displays the experiment status and LED state in real time.
